    Ada Pertanyaan?

    Jika jawaban atas pertanyaan Anda tidak tersedia di halaman ini, hubungi Account Manager.

    Apa saja yang termasuk dalam Langganan?

    Anda mendapatkan akses ke infrastruktur cloud terukur kami termasuk hosting, cadangan harian tambahan di dua benua, integrasi email, keamanan terbaik, pengawasan 24/7 dan pusat kontrol untuk mengelola sistem Odoo Anda. Pengalaman Odoo Anda ditingkatkan berdasarkan permintaan agar dapat memanfaatkan fitur baru sesuai yang Anda inginkan.

    Anda mendapatkan akses ke dukungan kami melalui email atau obrolan langsung dalam aplikasi. Tim dukungan kami tersedia dari Senin hingga Jumat, 24/5, dalam bahasa Inggris dan Prancis.

    Bagaimana cara mendefinisikan pengguna berbayar?

    Pengguna didefinisikan sebagai karyawan atau pemasok yang memiliki atau akan memiliki akses ke Odoo dalam mode buat dan/atau ubah. Pengguna eksternal (seperti klien atau pemasok pengguna portal) tidak dihitung sebagai pengguna. Misalnya:

    • Karyawan yang melaporkan pengeluaran atau tabel waktu akan dihitung sebagai pengguna;
    • Akuntan pihak ketiga yang menggunakan Odoo untuk mengerjakan hitungan Anda akan dihitung sebagai pengguna;
    • Pelanggan dan pemasok yang memiliki akses ke faktur online TIDAK dihitung sebagai pengguna;
    • Pengunjung situs web Odoo TIDAK dihitung sebagai pengguna.
    Dukungan apa yang diberikan oleh langganan mandiri (tanpa Paket Sukses)?

    Kami menyediakan dukungan fungsional dan layanan perbaikan bug pada fitur standar, 24/5, dalam bahasa Inggris melalui email. Konsultan khusus tersedia di Paket Sukses hanya.

    Mengapa beberapa aplikasi secara otomatis memerlukan aplikasi tertentu lainnya?

    Beberapa Aplikasi lebih diprioritaskan, yang berarti Aplikasi tersebut mengharuskan Aplikasi induk untuk beroperasi. Misalnya, eNiaga bergantung pada Aplikasi Situs Web. Untuk mengaktifkan eNiaga, Anda juga perlu menginstal Situs Web untuk mewarisi fungsi induknya.

    Dapatkah saya beralih dengan mudah dari paket Cloud Hosting (Odoo Online) ke paket lokal (Odoo Enterprise) atau sebaliknya?

    Saat menggunakan Odoo Online, Anda dapat men-download cadangan database kapan pun melalui pusat kendali. Anda yang mengendalikan data!

    Infrastruktur cloud kami berjalan pada versi terbaru Odoo dengan peningkatan reguler. Pengguna lokal harus menggunakan versi stabil terbaru Odoo. Jika Anda ingin beralih dari Odoo Online ke Odoo Lokal, waktu terbaik untuk melakukan perubahan adalah segera setelah perilisan versi stabil terbaru.

    Odoo juga menyediakan layanan untuk membantu Anda beralih dari Odoo Lokal ke Odoo Online secara gratis jika database lokal Anda menggunakan versi stabil terbaru. Untuk informasi selengkapnya atau jika Anda memiliki pertanyaan, hubungi kami.

    Bagaimana cara memilih Paket Sukses layanan penerapan yang tepat?

    Isi formulir di atas dan halaman akan menawarkan Paket Sukses terbaik sesuai kebutuhan Anda. Menurut pengalaman dengan proyek sebelumnya, kami menyarankan paket bisnis yang tepat, sehingga 80% pelanggan kami berhasil menerapkan Odoo untuk cakupan yang dibutuhkan, hanya dengan satu paket kesuksesan.

    Alasan paling umum pelanggan akan membeli Paket Sukses tambahan untuk penerapan awal mereka adalah masalah impor data (karena data penggabungan yang tidak terorganisir atau volume data yang terlalu besar untuk diimpor), penyesuaian yang tidak perlu (menghasilkan waktu tunggu yang lebih lama), atau perlunya pelatihan tambahan untuk karyawan sebelum penyelesaian penerapan.

    Some customers may also elect to purchase an additional Success Pack after implementation because they want to make their solution evolve over time as the company grows (support hours never expire).

    Dapatkah saya menggunakan nama domain sendiri untuk situs web?

    Tentu saja! Tim dukungan kami dapat memandu Anda selama penerapan untuk menyiapkan domain menggunakan Odoo berdasarkan cara Anda merencanakan untuk mengelola email menggunakan nama domain yang sama.

    Berapa harga aplikasi yang tidak tercantum di halaman ini?

    Beberapa Aplikasi di Odoo ditawarkan gratis seperti Survei, Diskusi, Catatan, Obrolan, dan Karyawan. Aplikasi lain disertakan seperti Blog, Forum, Obrolan Langsung, dan Slide dalam Aplikasi Situs Web dan tidak akan muncul di halaman ini.

    Mengapa harga penawaran Cloud dan Lokal sama?

    Biaya nyata untuk mengelola penerapan Odoo bukanlah biaya hosting (<1% dari laba kami), namun biaya karyawan terkait layanan pemeliharaan: dukungan, peningkatan, penggunaan, pemeliharaan server email/cadangan/database.

    Meskipun kami menetapkan biaya server tambahan untuk hosting pelanggan Awan, pelanggan yang tidak memiliki masalah konfigurasi email, selalu mendapatkan versi terbaru, tidak mengalami kesulitan upgrade, dsb.

    Jadi, meskipun terdengar paradoks, pelanggan Cloud mengeluarkan biaya lebih sedikit untuk pemeliharaan daripada pelanggan Lokal. Karena pelanggan Cloud biasanya memiliki pengalaman yang lebih baik (berfungsi dengan beberapa kali klik dengan infrastruktur yang disempurnakan), kami memutuskan untuk menawarkan hosting secara gratis.

    Berapa lama tawaran gratis berlaku?

    Penawaran gratis ini berlaku selamanya, untuk pengguna tanpa batas, selama Anda tidak menginstal aplikasi tambahan.
